# Jogo Snake

Este é o clássico jogo da cobrinha (Snake) desenvolvido por mim.

## Como jogar

- Use as setas do teclado para controlar a cobra.
- Coma a comida para crescer.
- Evite bater nas paredes e no próprio corpo.
